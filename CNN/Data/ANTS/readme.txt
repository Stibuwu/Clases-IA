Dataset

'Online Tracking of Ants Based on Deep Association Metrics: Method, Dataset and Evaluation' 
by Xiaoyan Cao, Shihui Guo, Juncong Lin, Wenshu Zhang, Minghong Liao


These two  Zip files contains train and test set used to obtain tracking results presented in Table 2 of the paper.

(1)bbox_train: For the offline pre-training, which can be done by using 'cosine_metric_learning'  repository.

(2)bbox_test:  For the online tracking, which can be done by using 'ant_tracking'  repository.



Noting: For specific training and testing details, please read the related repository.
